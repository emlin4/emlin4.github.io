Insertion

Insertion in a Red-Black Tree follows BST insertion rules but requires additional balancing steps:

  • New nodes are inserted as red.
  • If the parent is also red, rotations and color changes are applied.
  • The tree maintains its properties through rebalancing.

Deletion

Deletion requires careful handling to maintain Red-Black properties:

  • Standard BST deletion is performed first.
  • If a black node is removed, rebalancing is required to maintain black height.
  • Rebalancing includes color flips and rotations.

Search

The search operation follows standard BST rules:

  • Compare the key with the root node.
  • Traverse left if the key is smaller, right if larger.
  • Repeat until the key is found or a null node is reached.
