Insertion

Insertion in a BST follows the rule that smaller values go to the left subtree and larger values go to the right subtree.

Deletion

Deletion in a BST can be complex and has three cases:

  • Deleting a leaf node.
  • Deleting a node with one child.
  • Deleting a node with two children.

Search

The search operation in BST is performed recursively or iteratively by comparing values and navigating through the tree.

Traversal

There are three main ways to traverse a BST:

  • Inorder (Left, Root, Right) - Returns nodes in sorted order.
  • Preorder (Root, Left, Right) - Useful for copying the tree.
  • Postorder (Left, Right, Root) - Used in deleting nodes.
